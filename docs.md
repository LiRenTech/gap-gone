这是一个非常精简的专用于朗读文本剪辑的音频剪辑软件。

采用Tauri+React+Web Audio API实现。

# 产品愿景

为博客作者和内容创作者提供一个直观、简单的音频剪辑工具，专注于快速删除不需要的音频片段，无需学习复杂的时间线编辑。

# MVP 目标

开发一个纯网页应用，允许用户上传音频文件，按行显示波形图，通过点击选择删除片段，并导出处理后的音频。

# 核心功能

1. 音频导入与处理
   支持常见音频格式（MP3, WAV, M4A）

音频解码和波形数据提取

自动分割为固定时长的波形行

2. 波形可视化
   水平波形图显示（每行代表固定时长）一行一行的像乐谱布局一样铺满页面，再多的内容向下滚动

直观的振幅可视化

删除区域的视觉反馈（灰色覆盖）
左键任意波形位置，便可从任意位置开始播放。波形图上有一个红色竖线表示当前播放位置。

3. 剪辑功能
   点击选择删除区域（固定时长或自定义范围）

左键点击已经是修改播放指针位置了，所以编辑功能需要设计成鼠标中键和右键。
右键按下、滑动拖出一段选区、直到松开，形成一段线性区域。所形成的线性区域用半透明的灰色覆盖，表示即将删除的片段。
删除片段和删除片段之间如果出现重叠会自动合并。

需要注意鼠标向右拖出当前行的音轨区域即表示右侧结束区域在当前行的末尾。向左同理。

如果是中键拖动，则表示恢复一段选区，让灰色半透明区域恢复成未删除的保留状态，逻辑和右键同理。

目前先不做播放时自动跳过删除片段的功能，不排除以后可能会做。

4. 音频播放
   播放/暂停控制

播放进度同步显示

实时音频反馈

5. 导出功能
   导出为MP3格式

处理删除片段后的完整音频

保持原始音频质量
